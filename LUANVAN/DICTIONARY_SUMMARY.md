## 🎯 Cách sử dụng trong TTS

### Ví dụ 1: Du lịch quốc tế
```
Tôi bay từ <en>Vietnam</en> đến <en>Japan</en>. 
Ở <en>Tokyo</en>, tôi ăn <en>sushi</en> giá ¥5000.
Sau đó sang <en>Korea</en>, thăm <en>Seoul</en>.
```

**Đọc thành:**
```
Tôi bay từ việt nam đến giờ pàn.
Ở tô ki ô, tôi ăn xu si giá năm nghìn yên.
Sau đó sang cô ri a, thăm xê un.
```

### Ví dụ 2: Business Meeting
```
<en>Good morning</en> quý vị!
<en>Manager</en> từ <en>Singapore</en>, <en>Tokyo</en> và <en>NewYork</en> 
đã tham gia <en>meeting</en> qua <en>Zoom</en>.
<en>Sales</en> tăng 150%, <en>profit</en> đạt $500000.
```

**Đọc thành:**
```
gút mó ninh quý vị!
mé ni giờ từ xinh gà po, tô ki ô và niu ióc
đã tham gia mí tinh qua dum.
xây ồ tăng một trăm năm mươi phần trăm, pờ rô phịt đạt năm trăm nghìn đô la.
```

### Ví dụ 3: Đặt vé máy bay
```
Tôi muốn đặt vé <en>flight</en> từ <en>Hanoi</en> đến <en>Paris</en>.
<en>Airline</en>: <en>Vietnam Airlines</en>
<en>Price</en>: €800 hoặc 20000000đ
<en>Departure</en>: 10:30 ngày 20/05/2024
```

**Đọc thành:**
```
Tôi muốn đặt vé phờ lai từ hà nội đến pa ri.
e lai: việt nam e lain
pờ rái: tám trăm ơ rô hoặc hai mươi triệu đồng
đi pác chơ: mười giờ ba mươi phút ngày hai mươi tháng năm năm hai nghìn không trăm hai mươi bốn
```

## 📝 Notes

1. **Tag bắt buộc**: Phải dùng `<en>...</en>` cho tiếng Anh
2. **Case insensitive**: `<en>`, `<EN>`, `<En>` đều OK
3. **Nhiều từ**: `<en>New York</en>`, `<en>Good morning</en>` đều được
4. **Tên riêng**: Quốc gia, thành phố nên để trong tag `<en>`

## 🚀 Mở rộng thêm

Nếu cần thêm từ, có 3 cách:

### 1. Tạo file JSON riêng
```json
{
  "blockchain": "bờ lóc chen",
  "metaverse": "mé tà vớt",
  "nft": "en ép tì"
}
```

Sau đó load:
```python
normalizer = VietnameseTTSNormalizer(
    phonetic_dict_path="custom_words.json"
)
```

### 2. Sửa trực tiếp trong `normalize_text.py`
Thêm vào dictionary `default_dict`

### 3. Sử dụng LLM (auto-learn)
```python
from openai import OpenAI

llm = OpenAI(api_key="your-key")
normalizer = VietnameseTTSNormalizer(llm_client=llm)

# LLM sẽ tự động convert từ mới
```
